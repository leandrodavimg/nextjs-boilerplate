# Boilerplate NextJs 

```bash
yarn create next-app -e https://github.com/leandrodavimg/nextjs-boilerplate
```
## Contains
* NextJs
* TypeScript
* .editorconfig
* Eslint
* Prettier
* Git Hooks
* Husky
* Jest
* Styled-components
* Storybook
* Plop - generate files
* ChakraUI
